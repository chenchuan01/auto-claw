# AutoX Task Manager 安装指南

## 方式一：通过AutoX.js直接运行

### 步骤
1. 安装AutoX.js到Android设备
   - 从官方渠道下载AutoX.js APK
   - 安装并授予必要权限
   - 开启无障碍服务

2. 导入项目文件
   - 将整个`AutoXTaskManager`文件夹复制到手机
   - 在AutoX.js中点击"导入"
   - 选择`AutoXTaskManager`文件夹

3. 运行应用
   - 在AutoX.js中找到`main.js`
   - 点击运行
   - 授予必要权限

## 方式二：打包为独立APK（需要开发环境）

### 准备工作
1. 安装Node.js
2. 安装AutoX.js开发工具
3. 配置Android开发环境

### 打包步骤
```bash
# 1. 安装依赖
npm install -g autox-cli

# 2. 进入项目目录
cd AutoXTaskManager

# 3. 构建APK
autox build --platform android --output taskmanager.apk

# 4. 安装到设备
adb install taskmanager.apk
```

## 权限说明

### 必要权限
- **存储权限**: 用于保存任务数据和备份
- **网络权限**: 用于访问任务市场和更新检查
- **无障碍服务**: 用于执行自动化脚本
- **悬浮窗权限**: 用于显示执行状态

### 可选权限
- 通知权限：接收任务执行通知
- 位置权限：某些任务可能需要

## 常见问题

### Q1: 应用无法启动
- 检查AutoX.js是否已安装
- 检查是否已开启无障碍服务
- 检查存储权限是否已授予

### Q2: 任务无法执行
- 检查脚本语法是否正确
- 检查是否已授予必要权限
- 查看执行日志排查问题

### Q3: 无法访问任务市场
- 检查网络连接
- 检查防火墙设置
- 尝试切换网络环境

### Q4: 数据丢失
- 定期使用备份功能
- 不要手动删除应用数据
- 更新前先备份
